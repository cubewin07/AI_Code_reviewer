🚀 Getting Started Checklist

1

GitHub App Setup

Register a new GitHub App in your Settings page:

  • Set Homepage URL to this domain.
  • Activate Webhooks & paste the webhook target above.
  • Generate a Private Key and download the .pem file.
2

Configure Credentials

Place app credentials into your server environment variables:

  • Set GITHUB_APP_ID and GITHUB_WEBHOOK_SECRET.
  • Configure GITHUB_PRIVATE_KEY_PATH pointing to the .pem file.
  • Enter your NINE_ROUTER_API_KEY and LLM model choices.
3

Install and Run

Deploy the application and activate reviews:

  • Install the App on target repositories.
  • Ensure the JVM process runs via systemd or Docker.
  • Trigger auto-reviews by pushing a commit or creating a PR!
